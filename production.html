Scope summary (derived from repository)

Server: Node.js + Express + Mongoose (MongoDB). Frontend: static HTML/JS (Tailwind) for Seller Dashboard and Shop views.

  • User accounts: buyer/seller roles, signup/login, bcrypt password hashing, OTP and password reset via email.
  • Subscriptions and trials: plan gating (Free/Basic/Pro/Premium), trial flags, upgrade endpoints.
  • Products: CRUD, variants, color variants, specifications, categories/tags, featured, marketplace flags.
  • Assets: uploads via Multer; image persistence in Mongo (Image model) with /images/:id serving.
  • Orders: order creation, inventory checks per plan limits, buyer/seller HTML email notifications.
  • Cart and Wishlist: add/list endpoints; buyer summaries.
  • Coupons: create/list/update with plan restrictions; usage counters and savings.
  • Promotions: campaigns/ads with basic performance fields.
  • Payments: Paystack-oriented records (reference/status), list/update; currency helpers.
  • Analytics: seller endpoints for KPIs, sales over time, best hour, regional breakdown.
  • Dashboard UI: Tailwind-based seller portal for analytics, products, inventory, coupons, settings.
  • Shop/theming: static routes for shop, themes, layouts loader with allowlist.
  • Chatbot config: per-seller bot settings, training log, enable/disable.
  • Email service: SMTP transporter with environment-based configuration and templated messages.

Directories observed: models/, dashboard/, shop/, customer/, buyer/, seller/, review/, promotion/, public/, uploads/, themes/, layouts/, plus backend app.js and helpers.

Owner pricing strategy (recommended)

Option A – SaaS

  • Free: limited products, no coupons/analytics.
  • Basic: $9–$15/mo – inventory tracking, basic analytics.
  • Pro: $29–$49/mo – coupons, advanced analytics, premium gateways.
  • Premium: $79–$149/mo – priority support, custom domains, higher limits.

Add optional 0.5–2.0% transaction fee on gateway‑processed sales to subsidize costs.

Option B – One‑time License

  • Source code license: $60,000 – $90,000 (commercial use, limited support).
  • White‑label deployment: $3,000 – $8,000 per tenant setup + support retainer.

Risks and notes

  • Storing images directly in MongoDB increases cluster cost; prefer object storage (S3/Cloud Storage) for scale.
  • Harden auth flows: rate‑limits, session handling, CSRF/CORS review, production‑grade logging.
  • Add automated tests (unit/integration) to reduce QA cycles and regressions.
  • Payment verification webhooks for gateways should be fully implemented and tested.
  • Security review for file uploads and layout allowlists should continue before GA.
